# AI-Assisted Counseling Practice

A React + Vite application for practicing counseling skills through AI-assisted simulations, with feedback reports and a faculty review dashboard.

## Tech Stack

- React 19
- Vite
- React Router
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

This template provides a minimal setup to get React working in Vite with HMR and ESLint rules. Two official Vite React plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)
